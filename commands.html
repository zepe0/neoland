Windows

  • Comando para moverse entre carpetas: cd < path >
  • Comando para ver listado del directorio: dir
  • Limpia la ventana de la consola de Windows: cls
  • Cierra la ventana de la consola de Windows: exit
  • Muestra todos los comandos que hay disponibles, poniendo en cada uno una breve descripción en inglés: help
  • Te muestra la información de tu conexión, incluyendo tu dirección IP, la máscara de subred o la puerta de enlace por defecto: ipconfig
  • Borra el archivo o carpeta que le indiques : del
  • Crear nuevo directorio : mkdir nombre directorio
  • Crear nueva carpeta en el directorio: md
  • Renombrar archivos y carpetas : ren
  • Eliminar un directorio: rd

Linux/Mac

  • Lista el contenido del directorio indicado: cd
  • Abre el directorio especificado en la ruta: cd /nombre-directorio
  • Con el comando pwd podemos ver el nombre del directorio en el que nos encontramos en la terminal: pwd
  • Limpia la ventana de la consola : clear
  • Eliminar directorio : rmkdir nombre directorio
  • Crear nuevo documento: touch nombre archivo
  • Eliminar documento: rm nombre archivo
  • copiar archivo dentro de un directorio: cp nombre archivo direcion directorio
  • privilegios de superusuario (admin) : sudo
